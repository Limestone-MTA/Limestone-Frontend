Variables are assigned a type depending on the data they have.
 
Text variables contains data represented by text and can not be analyze with linear regression.
 
Continuous variables are numerical data with more than 15 different values.
 
Ordinal variables are numerical data with less than 15 different values.
 
Binary variables are data with only 2 values.
