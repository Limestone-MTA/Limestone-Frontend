Variables checked as :
  • Outcomes will be used as dependent variable;
  • Predictors will be used as explanatory variable;
  • Confounders will be systematically added as covariate.
  • Covariates will be assessed by Musical Chair® and some may be included in the regression model as covariates.
Note: The covariates selected by Musical Chair® will be shown in the results page.
 
Search variables supports regular expressions:
  • ^ag will show all variables starting with the string "ag" (age, agriculture...).
  • er$ will show all variables ending with the string "er" (gender, ).
  • gender|age will show all variables including either "gender" or "age".
  • * means any character.
  • ^((?!age).)*$ will show all variables without the word age.
