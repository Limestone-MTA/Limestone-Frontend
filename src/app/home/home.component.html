Welcome to Limestone

Limestone, a web interface for powerful association studies in large multidimensional data.

Limestone is an analytic software that allows performing linear and logistic regression from tabulated data. It includes automatic data pre-processing and quality control steps, but allows for advanced manual parametrization. The current version allows for standard multiple regression models, including confounding factors adjustement, and the advanced MusicalChair approach that leverages high multidimensional data to boost statistical power.
Details about the pre-processing, the quality control, and the modelling can be found in the Q&A.

Limestone regression model

Limestone is a framework for association studies from large multidimensional data. From the input data, the user select outcomes Y, predictor X and confounding factors F to be tested for association using a standard linear regression model:
f(Y)~X+F.
It also allows for advanced analyses using the Musical Chair algorithm*, which conduct a similar regression while selecting additional covariate to boost statistical power.
