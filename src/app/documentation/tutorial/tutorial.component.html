Getting Started with Limestone

In this guide, we'll walk you through the basic steps to get started with Limestone. By the end, you'll know how to import your data, generate visualizations and perform your first analysis.

Step 1: Create an new project

The first step to start your analysis on Limestone is to create a new project with your dataset.
You can get to the "Create a new project" page from the home or project list. Enter your project name and a description then you can upload a text, CSV or Excel file.

A brief explanation on the file format required is explained on this page but you can check the more detailed explanation here: File format
Once your project is ready you will be redirect to your project summary page.

Step 2: Start a new analysis

After you created a project you can start an analysis from the "Project summary" page by simply clicking on the "create an analysis" button.
An analysis has several steps :

  • I. Data transformation: Transform some of your variables, notably text variables.
  • II. Quality control: Apply quality filters on your variables.
  • III. Variable selection: Create your model by selecting which variable to include and which statistical test to perform.
  • IV. Results: Look into your results and download the complete table for further analysis.

You can choose to go through every steps for a precise control over your data or directly jump to the variable selection step by using the default parameters. To take a closer look at the "Data transformation" and "Quality control" steps please check the specific documentation about those steps here :

  • Data transformation
  • Quality control

Step 3: Visualize your data

At any steps of the analysis, you can click on your variables names to visualize more informations about it.
This will allow you to view the type, mean, variance and the distribution of your data.

Step 4: Variables selection

At this step you need to select which variables to add in your model.
Variables can be either :

  • Outcome: Variable to be predicted
  • Predictor: Variable that will explain the predictor
  • Confounder: Variable that influence both outcome and predictor causing a bias in the analysis. Adding them as confounder removes this bias.
  • Covariate: Variable that will be used by our Musical chair approach to improve statistical power.

For more details about this step don't hesitate to check the specific documentation: Variable selection.

Step 5: Results

Once the analysis has been completed you can with your results directly. For each couple Outcome-Predictor you have the results for 2 models:

  • The standard model using only the Outcome, Predictor and Confounders from your selection. (Y ~ X + F)
  • The Musical chair approach also adding the Covariates selected by the method to improve the statistical power of the analysis. (Y ~ X + F + C)

The results are composed of :

  • The coefficient of the regression "β" indicating the strength and direction of the association.
  • The p-value indicating if this association is significant, the threshold for significativness is indicated with the Bonferroni correction at the top of the page.

You have the option of downloading the result file to continue your analysis further.


To get more details about a specific association you can click on the loop on the left to open the complete details of the model, inculding all variables selected in the standard and Musical chair approaches.
