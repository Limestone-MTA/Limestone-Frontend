You can look at the details of your variables here.
Search variables supports regular expressions:
  • ^ag will show all variables starting with the string "ag" (age, agriculture...).
  • er$ will show all variables ending with the string "er" (gender, ).
  • gender|age will show all variables including either "gender" or "age".
  • * means any character.
  • ^((?!age).)*$ will show all variables without the word age.
